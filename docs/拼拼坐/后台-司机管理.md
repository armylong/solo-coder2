# 后台 - 司机管理（ppz-admin/drivers/）

## 一、页面概述

司机管理页面展示所有司机的列表信息，支持按状态筛选、查看车辆详情、封禁/解封司机等操作。

**访问路径**：`/static/ppz-admin/drivers/`

**前置条件**：管理员已登录

## 三、功能模块

### 3.1 状态筛选

| 筛选项 | 值 | 说明 |
|--------|---|------|
| 全部状态 | 0 | 展示所有司机 |
| 正常 | 1 | 仅展示状态正常的司机 |
| 已封禁 | 2 | 仅展示已封禁的司机 |

### 3.2 司机列表

**表格列**：

| 列名 | 说明 | 数据来源 |
|------|------|---------|
| UID | 用户ID | tb_ppz_cars.uid |
| 账号 | 登录账号 | tb_user.account |
| 姓名 | 用户姓名 | tb_user.name |
| 状态 | 司机状态 | tb_ppz_user.driver_status |
| 车辆信息 | 车辆型号列表 | tb_ppz_cars.car_model |
| 通过车辆 | 审核通过数 | 统计 count |
| 操作 | 封禁/解封/查看详情 | - |

**状态显示**：
| 状态 | 标签样式 |
|------|---------|
| 正常 | 绿色标签 |
| 已封禁 | 红色标签 |

### 3.3 封禁司机

**流程**：
```
点击「封禁」→ 弹出封禁确认弹窗
→ 填写封禁原因（可选）
→ 点击「确认封禁」→ 调用 banDriver API
  → 成功 → 刷新列表
  → 失败 → 显示错误信息
```

**封禁影响**：
1. 司机状态置为已封禁（driver_status=2）
2. 所有待审核和已通过的审核记录置为驳回
3. 司机无法添加/编辑车辆
4. checkDriver 返回 false

### 3.4 解封司机

**流程**：
```
点击「解封」→ 调用 unbanDriver API
  → 成功 → 刷新列表
  → 失败 → 显示错误信息
```

**解封说明**：
- 解封后司机状态恢复正常
- 但车辆审核状态不会自动恢复，需司机重新提交审核

### 3.5 查看车辆详情

**流程**：
```
点击车辆型号 → 弹出车辆详情弹窗
→ 展示车辆完整信息（型号、座位数、证件照片等）
```

### 3.6 分页

- 默认每页 20 条
- 支持上一页/下一页
- 显示总记录数

## 四、API 调用

| API | 说明 | 调用时机 |
|-----|------|---------|
| POST /ppz_admin/driverList | 获取司机列表 | 页面加载、筛选、翻页 |
| POST /ppz_admin/banDriver | 封禁司机 | 点击确认封禁 |
| POST /ppz_admin/unbanDriver | 解封司机 | 点击解封 |
| POST /ppz_admin/getCarDetail | 获取车辆详情 | 点击查看车辆详情 |

**请求参数**（driverList）：
```json
{
  "page": 1,
  "page_size": 20,
  "status": 0
}
```

**响应数据**：
```json
{
  "drivers": [
    {
      "uid": 100,
      "account": "zhang",
      "name": "张三",
      "status": 1,
      "ban_reason": "",
      "banned_at": "0001-01-01T00:00:00Z",
      "car_count": 3,
      "approved_car_count": 2,

          "car_id": 1,
```

## 五、数据查询逻辑

### 5.1 司机列表查询

```
1. 从 tb_ppz_cars 获取所有不重复的 uid
2. 按 status 筛选（通过 tb_ppz_user.driver_status 判断）
3. 按 uid 降序排列
4. 分页截取
5. 对每个 uid：
   a. 获取 tb_ppz_user 信息（司机状态、封禁原因等）
   b. 获取 tb_user 信息（账号、姓名）
   c. 获取车辆列表和统计
```

### 5.2 司机定义

系统中"司机"的定义是：**在 tb_ppz_cars 表中有记录的用户**。无论车辆审核状态如何，只要添加过车辆就算司机。
