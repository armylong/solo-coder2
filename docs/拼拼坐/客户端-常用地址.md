# 用户端 - 常用地址（ppz-m/address/）

## 一、页面概述

常用地址页面管理用户的出发地和目的地地址列表，支持添加、编辑、删除和排序操作。地址数据与高德地图集成，提供精准的地理信息。

**访问路径**：`/static/ppz-m/address/index.html`

**前置条件**：用户已登录

## 三、功能模块

### 3.1 地址列表

地址统一展示，不区分出发地/目的地：

**列表项展示**：
| 信息 | 说明 |
|------|------|
| 地址名称 | 用户自定义备注，如：我家、公司 |
| 地址详情 | 高德地图返回的详细地址 |
| 操作按钮 | 编辑、删除 |

**排序规则**：按 sort 字段降序排列（sort 越大越靠前）

### 3.2 添加地址

**流程**：
```
点击「+ 添加」→ 跳转地图选点页面（保存模式）
  → /address-map/index.html?mode=save
```

### 3.3 编辑地址

**流程**：
```
点击「编辑」→ 跳转地图选点页面（编辑模式 + 保存模式）
  → /address-map/index.html?mode=save&address_id=xxx
```

编辑模式下，地图选点页面会加载已有地址信息并回填。

### 3.4 删除地址

**流程**：
```
点击「删除」→ 确认弹窗 → 调用删除API → 刷新列表
```

**注意**：删除是软删除（status=0），数据不会物理删除

## 四、API 调用

| API | 说明 | 调用时机 |
|-----|------|---------|
| POST /ppz/ppz_map/addressList | 获取地址列表 | 页面加载 |
| POST /ppz/ppz_map/deleteAddress | 删除地址 | 点击删除确认后 |

**请求参数**（addressList）：
```json
{}
```

**响应数据**：
```json
{
  "list": [
    {
      "address_id": 1,
      "remark": "我家",
      "gaode_data": { ... },
      "sort": 2
    },
    {
      "address_id": 2,
      "remark": "学校",
      "gaode_data": { ... },
      "sort": 1
    }
  ]
}
```

## 六、地图选点页 mode 参数

地图选点页（address-map/）支持两种模式，通过 URL 参数 `mode` 控制：

| mode | 说明 | 选点后行为 | 使用场景 |
|------|------|-----------|---------|
| （默认） | 纯选点模式 | 选完直接 postMessage 返回 gaode_data | 订单表单选地址 |
| save | 保存模式 | 选完弹出备注弹窗，填写后 postMessage 返回 gaode_data + remark | 常用地址管理 |

**URL 示例**：
- 纯选点：`/address-map/index.html`
- 保存新地址：`/address-map/index.html?mode=save`
- 编辑已有地址：`/address-map/index.html?mode=save&address_id=xxx`
