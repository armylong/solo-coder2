## 页面说明

### 聊天列表页（主页）

路径：`/static/long-chat/`

- 展示所有聊天会话，按最新消息时间倒序
- 显示聊天名称、最后一条消息、时间、未读数
- 收到 WebSocket 推送时实时更新列表（未读+1、移到顶部）
- 右上角 🔍 搜索用户发消息（支持账号、昵称、手机号），➕ 弹出菜单（添加好友/创建群聊）
- 点击聊天项进入聊天详情页

### 聊天详情页

路径：`/static/long-chat/chat/index.html?chat_id=xxx&chat_type=1|2`

- 展示历史消息，自己发的绿色气泡，别人发的白色气泡
- 底部输入框发送消息，采用"本地先上屏 + 推送确认"模式：
  1. 点击发送 → 本地立即添加消息，气泡左侧显示转圈动画
  2. WebSocket 推送到达 → 替换临时消息，转圈消失
  3. HTTP 请求失败 → 转圈变红色感叹号
  4. 20秒超时未收到推送 → 转圈变红色感叹号，提示"消息发送超时"
- 收到 WebSocket 推送时实时追加对方消息
- 进入页面自动清零未读
- 超过5分钟的消息之间显示时间分隔线

### 通讯录页

路径：`/static/long-chat/contacts/`

- 顶部菜单：新的朋友（带未读数角标）、创建群聊
- 好友列表，每个好友有"发消息"按钮
- 右上角 ➕ 搜索添加好友（支持账号、昵称、手机号）

### 好友请求页

路径：`/static/long-chat/friend-req/`

- 展示待确认的好友请求
- 同意/拒绝按钮，操作后即时更新状态

### 创建群聊页

路径：`/static/long-chat/create-group/`

- 输入群名称
- 从好友列表勾选成员
- 已选成员显示在顶部标签区域，可移除
- 创建成功后跳转到群聊详情页

## API 调用

接口按 Controller 分组，通过 `Auth.fetchApi(baseUrl, action, params)` 调用。
路由映射规则：`ActionGetChatList` → `getChatList`（首字母小写，驼峰保留）。

| baseUrl | action | 说明 |
|---|---|---|
| `/long_chat/chat` | `getChatList` | 获取聊天列表 |
| `/long_chat/chat` | `createPrivateChat` | 创建单聊 |
| `/long_chat/chat` | `createGroup` | 创建群聊 |
| `/long_chat/message` | `listMessages` | 加载历史消息 |
| `/long_chat/message` | `sendMessage` | 发送消息 |
| `/long_chat/message` | `clearUnread` | 清零未读 |
| `/long_chat/friend` | `search` | 搜索用户 |
| `/long_chat/friend` | `listFriends` | 好友列表 |
| `/long_chat/friend` | `addFriend` | 添加好友 |
| `/long_chat/friend` | `listPendingRequests` | 待确认请求 |
| `/long_chat/friend` | `acceptFriend` | 同意好友 |
| `/long_chat/friend` | `rejectFriend` | 拒绝好友 |

## WebSocket 消息监听

```javascript
import { ws, WS_STATUS } from '/static/_common/ws.js';

// 监听新消息
ws.on('long_chat', (data) => {
    // data.chat_id     - 聊天ID
    // data.msg_id      - 消息ID
    // data.from_uid    - 发送者uid
    // data.sender_name - 发送者昵称
    // data.msg_type    - 消息类型
    // data.content     - 消息内容
    // data.created_at  - 发送时间
});

// 连接状态变化
ws.onStatusChange((status) => {
    // WS_STATUS.CONNECTING / OPEN / CLOSED
});

// 建立连接（自动带Token、自动重连、心跳保活）
ws.connect();
```

## 设计风格

- 主色调：微信绿 `#07c160`
- 页面头部背景：`#ededed`（微信灰）
- 单聊头像背景：`#576b95`，群聊头像背景：`#07c160`
- 自己的消息气泡：`#95ec69`（微信绿），对方消息气泡：`#fff`
- 未读角标：`#f44336`（红色）
- 最大宽度 600px，居中显示，移动端优先
